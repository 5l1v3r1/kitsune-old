Kitsune 
================================

This is a research project investigating frontends and backends to the Tapir
LLVM IR extensions.

Building with OpenMP -> Tapir Clang
===================================
    mkdir build
    cd build
    cmake ..
    make

Building with Flecsi+Kokkos -> Tapir Clang
==========================================
    mkdir build
    cd build
    cmake .. -DKITSUNE_CLANG
    make
